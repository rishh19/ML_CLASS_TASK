# 🧠 Neural Network – OR Gate

## 📌 Task Description

This task implements a simple neural network to learn the **OR gate logic** using Python.

The model uses sigmoid activation and gradient descent.

---

## 📁 Files Included

- 📓 **OR_NN.ipynb** → Jupyter Notebook implementation  
- 📄 **OR_NN_Algorithm.pdf** → Handwritten algorithm  

---

## 🧠 Model Overview

The neural network consists of:

- Input layer (2 neurons)
- Output layer (1 neuron)

---

## ⚙️ Mathematical Formulation

### Activation Function

σ(x) = 1 / (1 + e^(-x))

---

### Model Equation

y = σ(w·x + b)

---

### Loss Function

L = (1/n) Σ (y - ŷ)²

---

## 🔁 Training Process

1. Initialize weights  
2. Forward propagation  
3. Compute error  
4. Update weights  
5. Repeat  

---

## 🎯 Output

- OR gate predictions  
- Training loss graph
