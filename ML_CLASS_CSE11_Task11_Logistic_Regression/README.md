# 📊 Logistic Regression (Binary Classification)

## 🧠 Task Description
Implementation of Logistic Regression from scratch using Python in Jupyter Notebook.

---

## 📁 Files Included
- 📓 **Logistic_Regression.ipynb** → Python implementation of the model  
- 📄 **Logistic_Regression_Algorithm.pdf** → Handwritten algorithm steps  

---

## ⚙️ Algorithm Steps
1️⃣ Initialize weights and bias to zero  
2️⃣ Compute linear combination:  Z = wX + b  
3️⃣ Apply sigmoid function to get probability  
4️⃣ Compute prediction error  
5️⃣ Calculate gradients w.r.t weights and bias  
6️⃣ Update parameters using gradient descent  
7️⃣ Classify output using threshold **0.5**
