# 🧠 Single Layer Perceptron

## 📌 Task Description

This task implements a single layer perceptron for binary classification.

The perceptron learns using a step activation function and updates weights based on error.

---

## 📁 Files Included

- 📓 **Perceptron.ipynb** → Jupyter Notebook implementation  
- 📄 **Perceptron_Algorithm.pdf** → Handwritten algorithm  

---

## 🧠 Model Overview

The perceptron:

- Takes input features  
- Computes weighted sum  
- Applies step activation  
- Updates weights based on error  

---

## ⚙️ Mathematical Formulation

net = w·x + b  

y = sign(net)

---

## 🔁 Training Process

1. Initialize weights  
2. Compute output  
3. Calculate error  
4. Update weights  
5. Repeat
