# 🧠 McCulloch-Pitts Neuron

## 📌 Task Description

This task implements the **McCulloch-Pitts neuron model**, the simplest form of an artificial neuron.

It is a binary threshold unit used to simulate logical operations.

---

## 📁 Files Included

- 📓 **MCP_Neuron.ipynb** → Jupyter Notebook implementation  
- 📄 **MCP_Algorithm.pdf** → Handwritten algorithm  

---

## 🧠 Model Overview

The McCulloch-Pitts neuron:

- Takes binary inputs  
- Computes weighted sum  
- Applies threshold function  

---

## ⚙️ Mathematical Formulation

y = 1 if Σ(wᵢxᵢ) ≥ θ  
y = 0 otherwise  

---

## 🔁 Working Steps

1. Assign weights  
2. Compute weighted sum  
3. Compare with threshold  
4. Generate binary output  

---

## 🎯 Output

- Binary classification output (0 or 1)
