# 🧠 ADALINE Network – AND-NOT Function

## 📌 Task Description

This task implements an **ADALINE (Adaptive Linear Neuron)** network to learn the AND-NOT function using bipolar inputs.

---

## 📁 Files Included

- 📓 **ADALINE.ipynb** → Jupyter Notebook implementation  
- 📄 **ADALINE_Algorithm.pdf** → Handwritten algorithm  

---

## 🧠 Model Overview

ADALINE is a linear neuron model that:

- Uses linear activation  
- Minimizes mean squared error  
- Updates weights using gradient descent  

---

## ⚙️ Mathematical Formulation

Output:

y = w·x + b

Weight update:

w = w + α(t - y)x

Bias update:

b = b + α(t - y)

---

## 🔁 Training Process

1. Initialize weights  
2. Compute linear output  
3. Calculate error  
4. Update weights  
5. Repeat for epochs
