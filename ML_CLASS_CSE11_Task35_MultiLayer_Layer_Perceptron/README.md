# 🧠 Multilayer Perceptron (MLP)

## 📌 Task Description

This task implements a Multilayer Perceptron (MLP) using Python.

MLP is a neural network with hidden layers that can solve non-linear problems like XOR.

---

## 📁 Files Included

- 📓 **MLP.ipynb** → Jupyter Notebook implementation  
- 📄 **MLP_Algorithm.pdf** → Handwritten algorithm  

---

## 🧠 Model Overview

The network consists of:

- Input layer  
- Hidden layer  
- Output layer  

---

## ⚙️ Mathematical Formulation

Hidden layer:

a₁ = σ(XW₁ + b₁)

Output:

y = σ(a₁W₂ + b₂)

---

## 🔁 Training Process

1. Initialize weights  
2. Forward propagation  
3. Compute error  
4. Backpropagation  
5. Update weights
